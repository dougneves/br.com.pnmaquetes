Quem somos

A PN Maquetes é um estúdio de criação que desenvolve trabalhos em computação gráfica. Tem como seus principais produtos a Maquete Eletrônica 3D interna e externa, Tour Virtual em vídeo, Planta Humanizada para apresentação de layout e desenho técnico

Nosso diferencial é o atendimento personalisado, interando com seus clientes durante a execução do trabalho através de WorkingBlogs, um blog com acesso restrito para o cliente acompanhar todas as etapas durante a execução do trabalho.

Maquete 3D

A Maquete Eletrônica 3D é uma simulação de ambientes internos ou externos com o auxílio de computação gráfica.

Leva o cliente a experimentar antecipadamente um projeto antes de executá-lo, dando vida a algo que ainda está somente no papel.

Ajuda o projetista a identificar todos os desejos dos clientes, pois facilita a comunicação entre pofissional e leigo, diminuindo interferências que passam despercebidas em projetos 2D.

Auxilia também nas definições mais detalhadas do projeto, como cores, acabamentos, iluminação, etc.

A maquete eletrônica valoriza o projeto, com uma apresentação personalizada das idéias do projetista.

Video - percurso no ambiente

Seu cliente poderá admirar o projeto de forma mais envolvente, trazendo o cliente ainda mais para a realidade.

Tour Virtual 360

Permite apresentar uma cena a partir de um ponto fixo de onde é possível visualizar todo o ambiente. A ferramenta de zoom colabora para mostrar maiores detalhes.

Planta Humanizada

É uma planta layout com elementos de acabamentos e revestimentos do projeto, dando uma comprensão do ambiente de uma forma mais clara para o cliente, facilitando a apresentação.

Existe ainda a opção de Planta Humanizada 3D, onde a planta recebe elementos de perspectiva para uma comprensão tridimensional dos ambientes.

Projeto Técnico - CAD

Digitalização do projeto técnico (esboços, croquis), como plantas, cortes e elevações.
